About

  • Argument of all ephemerides is UTC
  • First data is for 00:00 UTC of the starting day.
  • Positions are expressed as geocentric apparent equatorial coordinates, unless stated otherwise.
  • All times (transits, eclipses, occultations etc.) are in UTC; this can be changed in "Settings".
  • Transits are computed for any given day as the first transit to occur after the given date. This means that they might occur in the following day.
  • Uses an Emscripten port of the AA+ library (version 1.80). More about Emscripten on their site.

Solar Eclipses

To obtain the value of a given Besselian element for a given time T, compute it in the following manner:

    t = T - T0
    A(t) = A0 + t * A1 + t2 * A2 + t3 * A3

where A is any of the elements from the tables below.

The values from the table are valid for a time interval from T0 - 3h to T0 + 3h.

x, y, l1 and l2 are in Earth radii.
μ and d are in sexagesimal degrees (°)
